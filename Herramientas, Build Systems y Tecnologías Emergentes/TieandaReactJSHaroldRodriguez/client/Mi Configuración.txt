********************** Angular 2 *************************
1.- Instalar Node.js
2.- Verificar si esta corriendo NodeJS
	node -v
3.- Verificar si esta corriendo npm
	npm -v
4.- Instalar Angular-CLI
	npm i -g angular-cli (Modo Global)
5.- Crear un nuevo proyecto de Angular
	ng new tienda (Dentro de la carpeta contenedora)
6.- Correr Servidor
	ng server (Dentro de la carpeta tienda)
